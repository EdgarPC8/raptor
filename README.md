# Raptor

Frontend compartido (marca Raptor).

```bash
cd raptor/frontend

# Modo invitado (SIN backend): Entrar como invitado → menú de módulos
npm run raptor
# → http://localhost:5175/raptor/

# App EdDeli (API :3001)
npm run eddeli

# App Store (API :3003)
npm run store
```

En modo invitado recorrés módulos/secciones (ficha demo). El tutorial spotlight viene después.

Envs: `.env.raptor`, `.env.eddeli`, `.env.store`.
